Um das Programm auszuführen muss im Terminal in den Ordner navigiert werden und unter ubuntu folgender Befehl eingegebenen werden:

swipl megaminx_algorithm.pl megaminx_goals.pl megaminx_rotations.pl mytest.pl

Das mytest Programm führt einen vorporgrammierten Testlauf aus. 
Sollen der Test geändert werden, kann im mytest einer der vier vorprogrammierten Testfällen verwendetet werden (Diese sind auskommentiert
und müssen nur wieder in den Code eingefügt werden). Alternativ ist es auch möglich den Würfel mithilfe einer eigenen Sequenz zu verdrehen und das 
Programm dies lösen zu lassen
